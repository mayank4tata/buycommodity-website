BCSPL Website V2.2.0
====================

This is a cumulative release over V2.1.4.

Retained from V2.1.4:
- Forgot Password and secure reset page
- Invitation-based Admin addition
- Users & Access
- Admin / Super Admin roles
- Disable/reactivate/reset/resend controls
- Access audit and server-side Edge Function security

Added in V2.2.0:
- One Catalogue & Pricing Master for Segment, Category, Product, Grade and Pricing
- Hidden system IDs; users work with normal commercial names
- Manual display-order controls at every relevant level
- Price Validity Days default of 1
- Editable, filterable HTML upload review
- Clear record-specific errors and downloadable error report
- Explicit-only pricing deletion and all-or-nothing database transaction
- Category Display Order
- Home search displaying matching Product pricing instead of opening Products

Deployment:
Read V2.2.0_DEPLOYMENT_GUIDE.md and run the V2.2.0 Supabase migration before
using Catalogue & Pricing Master in production.
